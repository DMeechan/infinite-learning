<p>How do you iterate through iterable stuff (like arrays or strings) in JavaScript?</p> 	<pre><code class="lang-javascript">for (let value of array) {     console.log(value); } </code></pre> 
<p>How do you iterate through objects in JavaScript?</p> 	<pre><code class="lang-javascript">for (let key in object) {     const value = object[key]; } </code></pre> 
<p>How do you reverse a string?</p> 	<pre><code class="lang-javascript">string     .split(&#39;&#39;)     .reverse()     .join(&#39;&#39;); </code></pre> 
<p>How do you get a character from a string?</p> 	<pre><code class="lang-javascript">// apple word.charAt(0) // returns a </code></pre> 
<p>How do you get a segment of an array or string?</p> 	<p><code>array.slice</code> returns a copy of the array (or string) segment; it does not mutate the original array</p> <pre><code class="lang-javascript">array = [1, 2, 3, 4, 5] array.slice(1, 3) // returns [2, 3] </code></pre> 
<p>How do you insert something (or multiple things) inside an array?</p> 	<p><code>array.splice(index, num_to_remove, ...items)</code> mutates the original array and will <code>add</code> and / or <code>remove</code> items</p> <pre><code class="lang-javascript">let array = [1, 2, 3, 4, 5]  array.splice(2, 0, &quot;dog&quot;, &quot;ate&quot;, 550, &quot;fish&quot;)  array === [1, 2, &quot;dog&quot;, &quot;ate&quot;, 550, &quot;fish&quot;, 3, 4, 5] </code></pre> 
<p>How do you remove (or replace) something(s) in an array?</p> 	<p><code>array.splice(index, num_to_remove, ...items)</code> mutates the original array and will <code>add</code> and / or <code>remove</code> items</p> <pre><code class="lang-javascript">let array = [1, 2, &quot;dog&quot;, &quot;ate&quot;, 550, &quot;fish&quot;, 3, 4, 5]  array.splice(2, 1, &quot;cat&quot;)  array === [1, 2, &quot;cat&quot;, &quot;ate&quot;, 550, &quot;fish&quot;, 3, 4, 5] </code></pre> 
<p>How can you divide a string into substrings (or even an array if you&#39;re a madman)?</p> 	<p><code>array.split(separator, limit)</code> takes a string, splits it up by the <code>separator</code> character(s) and then returns an array of the new split strings</p> <p>Both paramaters are <strong>optional</strong></p> <pre><code class="lang-javascript">const names = &#39;adam, bob, charlie, daisie, echo&#39;;  names.split() // returns [&quot;adam, bob, charlie, daisie, echo&quot;]  names.split(&#39;, &#39;) // returns [&quot;adam&quot;, &quot;bob&quot;, &quot;charlie&quot;, &quot;daisie&quot;, &quot;echo&quot;]  names.split(&#39;, &#39;, 3) // returns [&quot;adam&quot;, &quot;bob&quot;, &quot;charlie&quot;]  </code></pre>
